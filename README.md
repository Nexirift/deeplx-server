# DeepLX Server

A basic wrapper around DeepLX, based off of [Marie/deeplx-server](https://activitypub.software/Marie/deeplx-server). Now featuring latest updates and a new Docker image!

## Development

```
pnpm install
pnpm run dev
```

```
open http://localhost:3000
```

## Production

```
pnpm install
pnpm build
pnpm start
```

```
open http://localhost:3000
```

## Docker Image

```
docker build -t deeplx-server .
```
